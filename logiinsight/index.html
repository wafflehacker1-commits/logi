LogiInsight

An AI‑native smart webcam that offloads complex MediaPipe AI inference — Face Detection, Face Mesh (478 landmarks), and Hand Tracking (21 landmarks) — to an onboard NPU, dramatically reducing host CPU load.

What is LogiInsight?

Traditional webcams push all AI workloads onto the host machine. A Raspberry Pi 5 running MediaPipe face tracking can easily consume 85%+ CPU. LogiInsight solves this by embedding a dedicated Neural Processing Unit (NPU) inside the camera module itself.

The host simply connects over USB‑C, sends configuration commands, and receives structured JSON data — landmarks, bounding boxes, and confidence scores — with ~20% CPU usage.

Quick Start

Get up and running in seconds:

import cv2
from logiwebcam import LogiWebcamClient

# Open UVC camera (MJPG required for AI models)
cap = cv2.VideoCapture(0, cv2.CAP_V4L2)
cap.set(cv2.CAP_PROP_FOURCC, cv2.VideoWriter_fourcc(*"MJPG"))

with LogiWebcamClient(ip="192.168.1.10") as cam:
    cam.set_ai_config(enable_face_detection=True, enable_face_mesh=True)
    ret, frame = cap.read()
    data = cam.get_latest_ai_data()
    if data:
        for face in data["faces"]:
            x, y, w, h = face["bbox"]
            print(f"Face at ({x},{y}), conf={face['confidence']}")

cap.release()
