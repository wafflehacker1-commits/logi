Data Structures

Data Models

Detailed schemas for the AI output data returned by get_latest_ai_data(), including Face Mesh (478 3D landmarks) and Hand Tracking (21 3D landmarks) structures.

Top-Level JSON Schema

Every call to get_latest_ai_data() returns a dictionary with this structure (or None): 

{
  "timestamp_ms": 453404,
  "faces": [
    {
      "bbox": [956.0, 400.0, 264.0, 349.0],
      "confidence": 0.97,
      "landmarks": [
        [110.25, 210.50, 0.12],
        [115.75, 215.30, 0.08],
        ...
      ]
    }
  ],
  "hands": [
    {
      "bbox": [313.0, 385.0, 89.0, 137.0],
      "confidence": 0.87,
      "landmarks": [
        [60.25, 110.50, 0.00],
        [65.75, 115.30, 0.00],
        ...
      ]
    }
  ]
}
Key Type Description
timestamp_ms int Device-side epoch timestamp in milliseconds
faces list[FaceResult] Array of detected face objects (empty if none)
hands list[HandResult] Array of detected hand objects (empty if none)

Face Result Object

Each entry in the faces array contains:

Field Type Description
bbox list[float] (4) Bounding box as [x, y, width, height] in sensor pixel coordinates
confidence float Detection confidence score [0.0, 1.0]
landmarks list[list[float]] Up to 478 face mesh landmarks (empty if mesh disabled). Each point is an [x, y, z] array in pixel coordinates. 
# Accessing face data
data = cam.get_latest_ai_data()
if data:
    for face in data["faces"]:
        x, y, w, h = face["bbox"]        # [x, y, width, height] in pixels
        confidence = face["confidence"]
        landmarks = face["landmarks"]     # list of [x, y, z] arrays

        # Example: get nose tip (landmark index 1)
        if len(landmarks) > 1:
            nose_x, nose_y, nose_z = landmarks[1]
            print(f"Nose tip: ({nose_x:.2f}, {nose_y:.2f}, {nose_z:.2f})")

The face mesh model produces 478 three‑dimensional landmarks covering the full face topology — contours, eyes, eyebrows, nose, lips, and irises. Below is a schematic overview of key landmark regions.

Hand Result Object

Each entry in the hands array contains:

Field Type Description
bbox list[float] (4) Bounding box as [x, y, width, height] in sensor pixel coordinates
confidence float Detection confidence score [0.0, 1.0]
landmarks list[list[float]] (21) 21 hand landmarks. Each point is an [x, y, z] array in pixel coordinates (z is always 0.0).

# Mapping hand landmarks to gesture detection
# Each landmark is [x, y, z] in pixel coordinates
WRIST = 0
INDEX_TIP = 8
MIDDLE_TIP = 12
PINKY_TIP = 20

data = cam.get_latest_ai_data()
if data:
    for hand in data["hands"]:
        lm = hand["landmarks"]       # list of [x, y, z] arrays
        bbox = hand["bbox"]          # [x, y, width, height]
        conf = hand["confidence"]

        # Check if index finger is extended (tip above MCP)
        if lm[INDEX_TIP][1] < lm[5][1]:   # compare y coords
            print("Index finger is raised!")

        # Compute distance between thumb tip and index tip
        dx = lm[4][0] - lm[INDEX_TIP][0]
        dy = lm[4][1] - lm[INDEX_TIP][1]
        dist = (dx**2 + dy**2) ** 0.5
        print(f"Thumb-Index distance: {dist:.1f} px")

Hand Landmarks — 21 Points

The hand tracking model produces 21 three‑dimensional landmarks per detected hand. The diagram below shows the standard MediaPipe hand landmark topology.

Coordinate System — Point3D

Every landmark point in both face and hand results is a 3-element array [x, y, z]:

Index Axis Type Description
0 x float Horizontal position in pixel coordinates.
1 y float Vertical position in pixel coordinates.
2 z float Depth value. For face landmarks, relative depth from the face plane. For hand landmarks, always 0.0.

Note: Both bounding boxes and landmarks use sensor pixel coordinates, not normalized [0, 1] values. The sensor resolution is typically 3840x2160 (4K). To map these to your UVC frame resolution, scale by frame_width / 3840 and frame_height / 2160.
